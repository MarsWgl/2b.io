使用gitlab提交合并请求

发表于 2021-09-09
登录gitlab, 选择项目组
选择需要提交合并的项目
点击合并操作,进入界面
点击合并请求 New Merge Request 按钮, 选择源分支和目标分支
  • 表示将source branch 合并到 target branch
点击对比按钮进入下一个页面
提交之后到下一个页面, 等待request被merge 或者自己merge
解决merge冲突, 两种方式

  • 在线编辑, 点击按钮resolve conflicts , 会进入到在线编辑页面 , 冲突太多不建议使用这种方式

  • checkout到本地解决冲突, 点击merge locally, 按照文档流程操作, 建议使用phpstorm带有的解决冲突插件
